## Installation
- Create a new virtualenv
- Clone the repo
  ```bash
  git clone https://github.com/bdabve/shop
  ```
- install requirement
  ```bash
  pip install requirement
  ```
- Run
  ```bash
  python manage.py runserver
  ```
